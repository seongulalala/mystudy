# 정리
- t검정 : 두 집단의 평균 비교
  - 독립 t검정
    - 서로 다른 두 집단
  - 쌍체 t검정
    - 한 집단에서 전/후 비교

<br>

- ANOVA검정 : 두 집단 이상의 평균 비교
  - 일원분산분석
    - 독립변수가 1개
  - 이원분산분석
    - 독립변수가 2개
  - 반복측정분산분석
    - 동일한 대상의 전/후 비교

<br>

- f검정 : 두 집단의 분산 비교
<br>  

- Levene's test , Bartlett's test : 세 집단의 분산 비교
<br>

- 카이제곱 검정 : 두 집단(명목 척도 자료)의 빈도, 비율 비교

- 상관분석 : 두 연속형 변수 사이의 관계 확인
