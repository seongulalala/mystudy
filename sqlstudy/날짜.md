# 1 기본 함수
## 최근 N일 조회
NOW() : 현재 날짜 + 시간  
CURRENT_DATE : 오늘 00:00기준

INTERVEL N DAY : N일

```MYSQL
WHERE created_at >= NOW() - INTEVAL 7 DAY 
# 최근 7일 데이터
```

## 특정 기간 조회
```MYSQL
WHERE created_at BETWEEN '2024-01-01' AND '2024-12-31'
# 끝 날짜 제대로 포함하고 싶으면 <를 쓰자
```

## 날짜만 비교(시간X)
```MYSQL
WHERE DATE(created_at) = '2024-01-01'
```

## 요일함수
WEEKDAY(DATETIME) = 0 : 월 , 7 : 일
```MYSQL
WHERE WEEKDAY(created_at) IN (6,7)
# 주말만 조회
```

## 응용 함수
```MYSQL
이번 달 주문수

WHERE created_at >= DATE_FORMAT(CURRENT_DATE, "%Y-%M-01) 
created_at < DATE_FORMAT(CURRENT_DATE + INTERVEL 1 MONTH , "%Y-%m-01")
```

## %함수

|형태|사용법|
|---|-----|
%Y | 2025  
%y | 25  
%M | January  
%m | 01 ~ 12  
%d | 01 ~ 31  
%D | 1st, 2nd  

**주로 %Y-%m-%d를 쓴다**
