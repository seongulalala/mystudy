## 퍼널 대표 지표는 주로 비율 / 보조 지표는 절대값(시간, 수, 건수)
### 비율의 이유
#### 1. 규모 효과를 제거한다
절대값(시간, 수, 건수)은 항상 이렇게 공격받는다.

- “트래픽 늘어서 그런 거 아닌가요?”

- “세션 수 줄어서 평균 줄어든 거 아닌가요?”

비율은
- 분모로 규모를 고정
- 변화의 성격만 남긴다

#### 2. 비교가 가능하다
- 전월 vs 이번 달

#### 3. 판정에 쓰기 좋다

## 기본적인 비율들
### 1. 참여율
> 참여율 = 참여 세션 수 / 전체 세션 수  
판단 대상: 방문 품질  
분모 기준: 세션

### 2. 사용자 참여도
> 사용자 참여도 = 참여 사용자 수 / 전체 사용자 수  
판단 대상: 사용자 참여 여부  
분모 기준: 사용자

### 3. 이탈률(퍼널기준)
> 이탈률 = (이전 단계 세션 수 − 다음 단계 세션 수) / 이전 단계 세션 수  
판단 대상: 단계별 이탈  
분모 기준: 퍼널 설정 단위 (세션 또는 사용자)  
 
### 4. 재방문율
> 재방문율 = 재방문 사용자 수 / 전체 사용자 수  
판단 대상: 사용자 충성도  
분모 기준: 사용자

## 규칙
> 대표 지표는 판단용이어서 비율을 쓰고,
> GA4 비율 지표에서 분모는 ‘판단 대상이 되는 분석 단위’다.


### 기타

#### 이벤트 수와 비율은 다른 지표이기 때문에 이벤트 수 감소만으로 퍼널의 붕괴를 단정할 수 없다

#### 구매 퍼널은 한 번의 방문 내 행동 흐름이 중요하므로 세션 단위로 분석한다.

구매 전환은 하나의 방문 흐름 안에서 발생하므로, 이벤트 발생 순서를 동일 세션 내에서 보장하기 위해 세션 기준 퍼널을 선택한다.

#### 퍼널 전환율은 분자 : 다음 단계 / 분모 :이전 단계

#### 퍼널에서의 대표 지표 = 전환율 / 보조 지표 = 규모(이전 단계 세션 수 - 다음 단계 세션 수)


#### 퍼널 탐색에서는 이벤트 발생 순서가 보장되므로, (checkout → purchase) 전환율 등을 대표 지표로 설정하고 이를 채널별로 비교해 퍼널 이탈이 특정 유입 경로에 집중되는지 판단한다.
