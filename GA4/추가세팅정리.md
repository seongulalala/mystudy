# GA4 앱 설치
웹사이트는 GTM으로 관리했다면
앱은 Firebase Analytics SDK를 사용한다

### SPA
js(JavaScript)가 들어간 것은 SPA(single page application)이기 때문에
GTM설치 방법이 다르다
>SPA는 페이지 이동 시 전체 페이지가 다시 로드되지 않아, 표준 설치 방식으로는 첫 페이지 로드만 페이지 조회로 측정되고 내부 이동은 누락될 수 있습니다.
