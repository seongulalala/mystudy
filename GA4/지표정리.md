## 적합한 대표지표 
| 구분    | 적합한 대표 지표          |
| ----- | ------------------ |
| 볼륨 문제 | 수, 총량              |
| 퍼널 문제 | 전환 **비율**          |
| 구조 문제 | **강도 지표 (시간, 횟수)** |

## 볼륨
대표 지표 (절대량)

    사용자 수 (Total users)

    신규 사용자 수 (New users)

    세션 수 (Sessions)

    이벤트 수 (Event count)

보조 지표

    채널별 사용자 수

    디바이스별 세션 수


## 구조
A. 참여 “여부” 확인 (초기)

    사용자 참여도 (참여 사용자 / 전체 사용자)

    참여율 (참여 세션 / 전체 세션)

B. 참여 “강도” 확인 (핵심)

    세션당 평균 참여 시간

    사용자당 평균 참여 시간

    사용자당 이벤트 수

    사용자당 페이지뷰 수

    세션당 이벤트 수

👉 구조 문제의 진짜 핵심은 B

C. 지속성 / 리텐션

    재방문율

    N일 리텐션율 (코호트)

사용자당 세션 수

## 퍼널
대표 지표 (비율)

    단계 전환율

    퍼널 이탈률

보조 지표

단계별 도달 세션 수

    단계별 평균 소요 시간

    이전 단계 대비 다음 단계 잔존율

❌ 평균 시간 단독 사용 X
(전환 실패 위치를 설명 못 함)


| 비율 지표                         | 볼륨 문제 | 구조 문제        | 퍼널 문제       |
| ----------------------------- | ----- | ------------ | ----------- |
| **참여율** (참여 세션 / 전체 세션)       | X     | **2순위**      | 2순위         |
| **사용자 참여도** (참여 사용자 / 전체 사용자) | X     | **1순위(초기)**  | X           |
| **이탈률(퍼널)** ((이전−다음)/이전)      | X     | X            | **1순위(핵심)** |
| **재방문율** (재방문 사용자 / 전체 사용자)   | X     | **1순위(리텐션)** | X           |
