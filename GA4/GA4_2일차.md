### Q 우리 사이트 방문자가 요즘 어때?

다음 3가지를 고려해야 한다

1. 기간(일,주,월,년)
2. 용어(사용자 or 방문자)
3. 비교기준(비즈니스 의사결정 기준)

#### A "기간은 어떻게 할까요?" / 사용자를 보여드릴까요, 방문자로 보여드릴까요?" / "이 데이터 어디에 쓰시나요?"

# GA4 기본 구성 中 탐색
- 측정 기준 = 텍스트 = Dimension: 남자, 여자
- 측정 항목 = 숫자 = Metric: 남자 3명 , 여자 2명

### Q : 요즘 우리사이트 방문자 수가 어때?
- 방문자 = GA4에서는 세션

### Q : 아무것도 안하고 나간 사람들의 비율은?
- 이탈률

### Q : 8월에 갑자기 이탈률이 높아진 이유가 머지?
- 채널그룹
    - 세션 기본 채널 그룹 : 모든 사용자
    - 신규 사용자 기본 채널 그룹 : 신규 사용자
    - 기본 채널 그룹 : 모든 사용자

- 세션 기본 VS 기본
    - 세션은 한 사용자가 하루에 여러 번 사이트를 재방문해도 **첫 시점의 유입 채널이 고정값**으로 적용
    - 기본은 한 사용자가 하루에 여러 번 사이트를 재방문하면 방문할 때마다 채널을 판정

### Direct가 무슨 문제길래 이탈률이 높아졌지?
- 측정 기준 中 방문 페이지 / 이벤트 이름

### 홈페이지에서 이탈률이 높은데, 어느 기기에서 특히 높을까?
- 기기 카테고리 + 운영체제 및 버전

### 어디서 왔을까?
- 소스/메체 : 고객들의 유입 경로

### 고객이 무엇을 했지?
- 방문 페이지
