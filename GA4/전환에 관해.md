## 전환 설정에 관해

거시 전환 : 비즈니그의 주된 목표를 충족하는 핵심 행동 (ex: 구매)

미시 전환 : 비즈니스의 주된 목표에 가까워지도록 유도하는 작은 목표 (ex: 이메일 쿠폰 신청, 신상품 알림 신청)

> 전환을 생각할 때는 미시 전환을 잘 고려해야 한다  
ex)   
넷플릭스 거시 전환 : 구독(후행지표)
넷플릭스 미시 전환 : 시청 시간(선행지표)

## 전환율
> GA4에는 2가지의 전환율 측정항목이 있다

- 사용자 中 사용자 전환율(측정 항목 이름 = 사용자 주요 이벤트 비율)  
= 주요 이벤트를 1회 이상 발생시킨 사용자 수 / 전체 사용자 수

- 세션 中 세션 전환율 (측정 항목 이름 = 세션 주요 이벤트 비율)   
= 주요 이벤트가 1회 이상 발생한 세션 수 / 전체 세션 수

- 전환이 주요 이벤트로 이름이 바뀜
- EX) 전환 = 주요 이벤트, 세션 전환율 = 세션 주요 이벤트


> 한 명의 유저는 여러 번의 세션을 실행할 수 있으니 주로 사용자 전환율이 세션 전환율 보다 높게 나온다
