# 탐색함수
- **Row들을 탐색하며 값을 찾는 함수**
- 정렬이 항상 필요

- 순서
    > Partition by > Order by > 찾기

## LEAD(후속 행 찾기) / LAG(이전 행 찾기)
`LEAD(컬럼) OVER (PARTITION BY 컬럼 ORDER BY 정렬할컬럼)`

`LEAD(컬럼, 숫자) OVER (PARTITION BY 컬럼 ORDER BY 정렬할컬럼)`

`### 여기서 숫자는 몇 번쨰 다음(이전) 행을 가져올지`

## FIRST_VALUE(첫번째 깂 찾기) / LAST_VALUE(마지막)
```
FIRST_VALUE(컬럼) OVER (PARTITION BY 컬럼 ORDER BY 정렬할컬럼
ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
)
```

#### FIRST_VALUE와 LAST_VALUE는 NULL을 포함해서 연산한다
#### NULL을 제외하고 싶다면
`FIRST_VALUE(컬럼 IGNORE NULLS) OVER (ORDER  BY 컬럼)`
